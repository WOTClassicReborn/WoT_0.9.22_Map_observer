# WoT_0.9.22_Map_observer

This mode allows you to explore any map in version 0.9.22 with any tank of your choice.


--HOW DO YOU DO THAT--
1- Download this mod
2- Place the contents in the "mods" folder for the correct version
3- Download PjOrion: https://drive.google.com/file/d/1659pR-ce4yAVQ-LOyUpEaNEmRNSw7Pp7/view?usp=sharing
4- Open the mod_observer.py file in the mods\0.9.22.0\res\scripts\client\gui\mods folder
5— Edit the tank on line 193, which looks like this: self.selectedVehicleTag = 'sweden:S16_Kranvagn'
6- Enter the correct country: Germany: germany
                              Soviets: ussr
                              America: usa
                              Sweden: sweden
                              Japan: japan
                              China: china
                              Czech Republic: czech
                              France: france
                              Britain: uk
                              Poland: poland
7- Then enter the correct vehicle ID, which you can find in res/packages/vehicles_level_10.pkg; it looks something like this: Cz04_T50_51
8— Save the .py file
9- Compile the .py file into a .pyc file. First, make sure you have selected the correct Python version: Terminal> Settings> Basic Setup> and in the Python field, select python27.dll, click Connect, OK, and then click Terminal> Save As> and select Python bytecode files
10— Return to the mods\0.9.22.0\ folder, right-click on "res," select 7-Zip, and the "Add to Archive..." option will appear... Enter the full name izeberg.observer_1.0.4.2.wotmod and save it
11- You can start the game
--END--
